# Project Objective

Create a real estate sales application called Luxe Estate, using Nuxt4, vue 3, TypeScript and Tailwind.

The project is already created and has Tailwind installed, but it lacks styles and functionalities. The goal is to implement the design and basic functionalities of a modern, premium, and minimalist real estate app, following exactly the designs presented in the provided references located in the "Antigravity" folder within the project.
